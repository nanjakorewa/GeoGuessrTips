電柱

中南米

    四角と丸の電柱。下が白色に塗られているものがある。四角い電柱の区切り位置は広め。

    木製のもの or 八角形のものが多い。

    四角いポール。色が塗られているときもある。

    電柱は道路側に３～４本棒が付きだしているものがある。
