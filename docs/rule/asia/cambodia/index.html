カンボジア

国・地域の見分け方

  • 車は右側通行
  • ドメインは.kh
  • 白と赤に塗られたボラードがある
  • 仏教に関係する建築物が多い
  • クメール語（ភាសាខ្មែរ）が広く用いられている
  • カンボジアの農村部は高床式住居が多い
  • パラソルなどにビールブランドのLEO・Kingdom・Angkorのロゴがある
  • ちいさな祠のようなものがあって家のある方角に向かって置かれていることが多い
  • パラソルの下でビン入りの飲み物がよく売られている

見つかる標識

    クメール語（ភាសាខ្មែរ）が公用語 。白と赤に塗られたボラードがある 。

    By Dmitry Makeev - Own work, CC BY-SA 4.0, Link

    カンボジアの農村部は高床式住居が多い 。カンボジアは熱帯モンスーン気候であり年間を通して高温多湿。高床にすることで昼は暑さ対策、夜は牛や水牛が休むスペースになる。雨季の大雨からの浸水対策にもなるらしい (参考文献 2021．April1 在仙台カンボジア王国名誉領事館 瓦版 NO.6)。

    カンボジアで知名度の高い企業が書かれたパラソルや看板が多い 。通信はBeeline、清涼飲料水はコカ・コーラ、ビールはLEO・Kingdom・Angkorなど、Angkorのロゴにもアンコールワットが描かれている (参考文献 カンボジアのビール事情)。郊外ではこのようなパラソルの下で瓶入りの飲み物が売られていることが多い 。

    仏教徒が多く仏教に関係する建築物が多い。また土地神や先祖神を祀っている祠があって家の方に向かって置かれていることが多い (参考文献 vol.348 身近な家の神様たち【カンボジア】)。写真の金色の小さい祠がそれ。

    こういう感じの木がある (参考文献 オウギヤシ)。 ブラジルなどの暖かい地域全域似に似た見た目の木があるがアジアではカンボジアが特に多い。土の色も特徴的 。

    州・地域の絞り込み

    • コーン畑や田んぼは東側と海沿いにはほとんど存在しない (参考文献 USDA Cambodia Production)
      • 畑が少ない北や東の地域
      • 畑のある中央地域
      • 畑が少ない海沿いの地域
    • ボラードを見つけて道路番号を探したあとで両面に書いてある町名を探す 。そのあと距離の比率をみておよその地域を絞り込む。

      コーン畑や田んぼは東側（7番の道路の東側）と海沿いにはほとんど存在しない (参考文献 USDA Cambodia Production)。

      ボラードの側面に道路番号が、両面に町までの距離が書いてある。運よく見つけられたらかなり距離を詰めることができる。

      都市・町の絞り込み

      • Battambang西の タイ国境付近の道では険しい岩山が見えることがある
      • Tonlé Sap湖ではボートハウスの村がある (参考文献 បឹងទន្លេសាប)
      • Kampot旧市街には フランス植民地時代の建築が多く残っている (参考文献 カンポット)
      • Siem Reapにはアンコールワットやバイヨン寺院がある
      • Preăh Seihânŭは輸出入の拠点であり中国からの投資により発展している (参考文献 Preah Sihanouk province)。高い建物も存在する 。

        西の国境付近の道では険しい岩山が見えることがある 。上の農業分布でBattambangの西の空白になっている部分は岩山が多い。

        トンレ・サップ湖の上も出ることがある。ボートハウスの村が湖面にあり独特な建築が見られる 。

        カンボジアの都市部ではフランス植民地時代の建築が多くみられる。Kampotは重要な港町で特に多く洋風の建築が残っている。

        By Hans A. Rosbach - Own work, CC BY-SA 3.0, Link

        Siem Reapにはアンコールワットやバイヨン寺院がある 。

        アンコールワットのあるシェムリアップでは条例により高い建物は作れない (参考文献 2021．April1 在仙台カンボジア王国名誉領事館)一方でPreăh Seihânŭは４階建て以上の建物が存在する 。石油ターミナルもあり物流拠点になっている。
