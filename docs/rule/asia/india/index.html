インド

国・地域の見分け方

  • 言語はヒンディー語など（地域によって違う）
  • ドメインは.id
  • インドは都市圏でも住所システムが完全には整備されておらず、商品が届かないことがある(from 『インドEC市場調査報告書 - 日本貿易振興機構』)。そのため確実に郵便が届くように住所を看板などに明記している店が多い？日本と逆で、末尾ほど大きな地域名。

    バーンスリーはインドのヒンドゥスターニー音楽でよく使用される楽器。

    州・地域の絞り込み

    • 地域ごとに言語が違うので文字の形で見分けられる(by usumdensetu )
    • デリーなどの都心はセクターやブロックの名前でさらに地域を絞り込める、ただし同じ名前のセクターが違う場所にもあるので注意

      アフマダーバード周辺。ちなみに階段井戸もアフマダーバード周辺に多い。

      カルナータカ地方とベンガルール周辺に話者が多い。出典：(from Wikipedia 『カンナダ語』)

      ヒンディー語・ヒンドゥスターニー語はデリーを含む地域で話される自然言語。

      スリランカの海岸沿いに多い。
