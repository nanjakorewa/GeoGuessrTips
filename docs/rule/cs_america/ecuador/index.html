エクアドル

国・地域の見分け方

  • ドメインは.ec
  • 言語はスペイン語
  • 標識の裏が凹凸のないフラットな感じのものが多い(by Geotips )
  • タクシーやバスのような商業車はナンバーの全体か上部がオレンジ色
  • ボラードに２つ反射板が巻かれている・２つ反射板が付いているならエクアドルかも

見つかる標識

    ボラードはいろいろな種類があり一貫性がない。反射板が付いている見慣れない形のものはエクアドルかも？

    標識の裏がフラットな感じ (by Geotips )でシンプルな□や○の形の鋼管 や凹みのある鋼管 を使って立てられている。 コロンビアは白いポールで裏側に十字があり、 ペルーは白黒ストライプの太い棒が多い。 ボリビアは太い製材が使われている。また、エクアドルのガードレールは二重になっていることが多い(by Geotips )。

    By Paulo JC Nogueira, CC BY-SA 3.0, Link, 画像加工あり

    タクシーやバスのような商業車はナンバーの全体か上部がオレンジ色。緑色のステッカーがあるとキト(by youtubeのコメント欄 ギガバイトさん)。

    By Zcarstvnz - Own work, CC BY-SA 4.0, Link

    コンクリート製の道路も比較的多く見られる (by Geotips )

    シェブロン＞＜

    ボラードは色々なものがあるが丸くて見たことが無い感じのボラードはエクアドルかも？

    赤や白のぼかしにアンテナがちょっと見える。

    • https://goo.gl/maps/GenXMQanMSXWMaj5A

    州・地域の絞り込み

    • ズームすると県境に県名が表示される
    • アブラヤシのプランテーションがあるのは南の方に限られる
    • ガラパゴス諸島が出ることがある・亀が山の上に居たら ガラパゴス諸島

      看板などから県名が見つかったら県名で場所を絞り込めるかも（画像出典：©2023 Google Map, 2023/4/28に利用, Google マップ 一般的なガイドライン）

      赤い丸の箇所にはアブラヤシのプランテーションがあるかも。特にコロンビア国境付近の道は木が多い。 (参考文献 USDA - International Production Assessment Division Reports)

      都市・町の絞り込み

      • タクシーに緑色のステッカーがあるとキトかも(by youtubeのコメント欄 ギガバイトさん)
