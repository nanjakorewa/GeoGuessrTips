ブラジル

国・地域の見分け方

  • ドメインは.br
  • 言語はポルトガル語
  • 標識の裏側が黒いものが多い
  • 家のアンテナが特徴的で透明なパラボラアンテナ（タイなどにも似たようなものがある）
  • ナンバープレートの上側が青色の時がある(by Geotips )
  • トラック・バス・タクシーの車両のナンバープレートが赤かったり文字が赤かったりする。
  • 電柱の溝に仕切りがある電柱を使っている。仕切りがない電柱はチリの可能性が高い。

    標識の裏側が黒いものが多い

    家のアンテナが特徴的、透明なパラボラアンテナ、ただしこれだけでは確定できないと思う

    トラック・バス・タクシーの車両のナンバープレートが赤かったり、文字が赤かったりする。最近の車ほど上が青いことが多い。画像は一部加工あり。

    By Olympiobr - Own work, CC BY-SA 3.0, Wikimedia Commons

    左がブラジル、右がチリの電柱。チリの電柱には仕切りが無いことがわかる。

    南米最大規模の銀行、Banco Itaúのオレンジ色の看板がたくさんある。ガソリンスタンドに南米石油最大手ペトロブラス（緑色のロゴ）のものが多い。

    州・地域の絞り込み

    • 地域ごとに植生と土の色が異なる
      • Amazônia：湿度の高い赤道気候でアマゾンの森林が存在する
      • Cerrado：畑が多い地域も木がたくさんある地域もあり判別が難しい
      • Mata Atlântica：ブラジルの大西洋の海岸線15州以上に広がって分布する森林
      • Caatinga：乾燥した土壌であり白っぽい土壌が含まれることが多い
      • Pampa：ウルグアイに近い雰囲気で農牧地がある
      • Pantanal：パラグアイ北部の河川による沖積平野だが道がほとんど無い
    • 特定の地域や州が分かる木
      • ババスの木：ピアウイ州とマラニョン州(from 『【Geo Cup】日本最強による大会フル解説！！【翻訳】【Geoguessr】』)

      AmazôniaとPantanalは道が少ないけれど、他の地域は多くの道が存在する。（画像出典：By Brazil Travel, CC BY-SA 4.0, Wikimedia Commons）

      地域ごとに植物も異なる。海から吹く湿った風の影響を受ける。他の地域とは森の見た目が異なる。ヤシや松のようなものが見える時がある気がする。(参考文献 ブラジル大西洋岸森林（マタ・アトランティカ）の生態及び保全)。

      • 電話番号のエリアコードで範囲を絞ることができる
      • BR-XXXはブラジル国道。BAでないものは州の名前かも。州名はマップをズームアウトすると確認できる。
      • 電柱にブラジルの電力会社名が書かれていることがある。ニューヨーク証券取引所に上場しているある程度有名（？）な会社は以下３つ。
        • ブラジル電力(Eletrobras)→ブラジル全域どころかアフリカにまで進出している
        • パラナ電力(COPEL)→パラナ州
        • ミナスジェライス電力(CEMIG)→ミナスジェライス州
        • 黄色く塗られた領域がある場合はマラニョン州(from 『【Geo Cup】日本最強による大会フル解説！！【翻訳】【Geoguessr】』)

        1がサンパウロ、2がリオデジャネイロ。（画像出典：By Magno Brasil - Own work, CC BY-SA 4.0, Wikimedia Commons）

        COPEL=パラナ電力の文字がうっすらと刻まれている
        CEMIG=ミナスジェライス電力の文字がパイプに書かれている
