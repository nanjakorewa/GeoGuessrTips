国の見分け方

  • ナンバープレートの上側が青色の時がある (by Geotips )
  • 南米最大規模の銀行、Banco Itaúのオレンジ色の看板がたくさんある
  • ガソリンスタンドに南米石油最大手ペトロブラス（緑色のロゴ）のものが多い
  • ドメインは.br
  • 家のアンテナが特徴的、透明なパラボラアンテナ
  • 標識の裏側が黒いものが多い

州・地域の見分け方

  • 道路番号を見ればおよその地域がわかる、BR-XXXはブラジル国道。
  • BAでないものは州の名前を表している、州を特定したらその中で番号を探すと道が見つかる。
