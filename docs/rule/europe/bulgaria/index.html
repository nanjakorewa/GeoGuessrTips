ブルガリア

国・地域の見分け方

  • ドメインは.bg
  • 横断歩道の標識の線の数は4本
  • 公用語はブルガリア語（български）でキリル文字を使用する
  • 地域名などの看板が基本的に２言語で書かれている
  • 「Ъ」の文字はブルガリアとロシアでしか使われない (参考文献 Ъ)
  • ブリキ製のようなゴミ箱が通りや家の前に置いてある(by Geotips )
  • なんか全体的に天気が曇りっぽい

    タイヤのついたブリキ製の丸いふたのゴミ箱が通りや家の前に置いてある (by Geotips )。他の国にもあるときはあるのでこれだけで判断しないほうがいいかも。

    路面のコンディションが非常に悪い。現地の人が路面状況マップを作って悪い道を避けるレベル (参考文献 Bulgarian Road Quality Map)。一人当たりのGDPは2022年時点ではEUで一番低い (参考文献 外務省 - ブルガリア共和国基礎データ)ためか、家やインフラはボロボロなものが多い。
