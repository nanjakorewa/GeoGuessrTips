フランス

国・地域の見分け方

  • ドメインは.fr
  • 電話番号が2つの数が5個並んでいる
  • 「D653」と言った感じのDで始まる道路番号の黄色いプレートがある
  • フランス語の「â，î，û，ê，ô」とアクサンテギュ（「e」に点をつけた文字）が特徴的
  • 電柱に青いプレートが貼ってあることがある(from 『Geoguessr Tips: Guide to European Utility Poles』)
  • 通り名の看板に「Rue」と書いてあればフランス語圏の可能性が高い

見つかる標識

    電話番号が2桁が5つ並んでいる。はじめの番号（いわゆる市外局番に相当するもの）で地域もわかりパリが 01だ。電柱やポールに青いプレートが貼ってあることがありこれも特徴的。

    ボラードは丸く太い。赤と白のものがあり、先がとがっている。

    ナンバープレートは両サイドに青の時がたまにあるが片方だけ青の場合が多い。「D653」と言った感じのDで始まる道路番号の黄色いプレートがある。

    By I, Bzh29, CC BY-SA 2.5, Wikimedia Commons

    Système Uはフランスのスーパーマーケット組合 (from Wikipedia 『Système U』)。強くなるとHyper Uになる。

    州・地域の絞り込み

    • 電話番号の先頭の2桁で地域がわかるかも。01はパリ周辺、あとは時計回り。離島（コルス島）は04を覚えておけば気合で思い出せる？
    • ドイツ語が混ざっていたら県境の近くを検討してみる(by now_and_jenn )
    • レンガが多かったらベルギー近くを検討してみる
    • ヨーロッパの農業分布
      • 菜種の畑は北部に多い
      • ひまわりの畑は北部沿岸部には少ない

      電話番号が2桁が5つ並んでいる。はじめの番号（いわゆる市外局番に相当するもの）で地域もわかる。(画像：CC BY 3.0. Telephone numbers in France. (2023, March 30). In Wikipedia. )。パリが 01 で、あとは時計回りなので他の国と比較すると覚えやすい。

      都市・町の絞り込み

      • パリの場合は区画の番号が時計回り、店の住所を見れば直ぐに場所が分かる(by now_and_jenn )
      • ヴェルサイユ宮殿の中も出題される

        パリの行政区はル・ルーヴルの１からスタートしてらせん状に数が増える。

        ThePromenader, user:yonidebest added the numbers - Image:Paris arr jms.gif, CC BY-SA 3.0 - Link

        通りの交差点近くの建物の壁に『●●ARR, RUE ◆◆』と書かれたプレートが張られている。下のプレートの場合はパリの6番目の区画に行って「RUE ◆◆」の通りを見つければok。
