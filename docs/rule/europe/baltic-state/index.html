左からリトアニア・ラトビア・エストニア

  • リトアニア：オレンジのボラード・斜めの距離看板・縁のあるシェブロン
  • ラトビア：縁のないシェブロン・道路に平行な距離看板・「ī」の文字
  • エストニア：縁のない速度制限の看板
