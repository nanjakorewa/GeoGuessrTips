ラトビア

国・地域の見分け方

  • ドメインは.lv
  • 横断歩道は５本
  • バスの窓が５つ
  • 標識に縁がある
  • 通り名の看板に「～iela」と書いてある
  • 数字が書かれた標識が道路に対して平行
  • ボラードを真上から見ると）の形
  • リトアニアと速度上限の標識などを比較したとき赤い線が太い
  • 言語に特徴がある
    • Ļ・ļの文字が特徴的
    • 「Rīga」の「ī」のように母音に￣の記号が付いている
    • Ł・łならば ポーランドかも
    • Ș・șならば ルーマニアかも

    「ī」のように母音に￣の記号が付いている文字。

    電柱が丸いものが多く電柱の上の部分にフックみたいな形のものがついている 。似たような形の電柱は ブルガリアや クロアチアにもある。

    バルト三国の中ではラトビアのみ横断歩道の線の数が５本でバスの窓も５個。そして道路の横の標識の向きが道路に平行。

    リトアニアと速度上限の標識などを比較したとき赤い線が太い 。縁だけ比較した下の図は左からエストニア・ラトビア・リトアニア。

    ボラードは文字が書いてあることもある。上から見ると『）』の形をしている 。下の図は左からリトアニア・ラトビア・エストニア (参考文献 画像１枚でバルト三国見分けたい)。

    シェブロンの色や枠がバルト三国で異なる。下の図は左からエストニア・ラトビア・リトアニア (参考文献 How to tell Baltic countries apart 2.0: Expanded edition)。
