オランダ

国・地域の見分け方

  • ドメインは.nl
  • ナンバープレートは黄色、前後を見てこの色ならオランダになる
  • 標高について国土の4分の1が海抜0m以下で最高地点も約333m、平らで高い山が無い
  • 家がレンガ造りのものが多く、最上階付近の壁に棒や突起物がある時も。
  • 自転車を走る道路（Fietspad）が国中に整備されていて茶色っぽい路面か赤レンガで舗装されているケースが多い。
  • ルクセンブルグと比べ平坦で、水路が多く、家が全体的に赤っぽく、自転車用の道路数が多いように思える 家の見た目 > オランダ

見つかる標識

    ナンバープレートは左側が青色で背景が黄色であり前後を見てこの色ならオランダになる

    Gert Kalkman - Public road, パブリック・ドメイン, Wikimedia Commonsによる

    家がレンガ造りのものが多く、稀に最上階付近の壁に紐をひっかける棒があることもある。貿易産業が盛んなころに倉庫だった建物は荷物を吊り上げるためにこの棒が付いている。現在でもこの金具にロープを吊るして家具を吊り上げているらしい。(参考文献 あの国で留学 > 世界の家【オランダ編】)

    標識を後ろから見たときに表側の色が裏の縁取りに見える。

    道路番号が書いてある

    SPARはアムステルダムに拠点がある小売業者（店自体はオランダ以外にもある）

    州・地域の絞り込み

    • キュラソー島（カリブ海のオランダ領の島）が出ることがある キュラソー
