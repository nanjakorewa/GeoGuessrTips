国の見分け方

  • ドメインは.es
  • 歩行者注意の看板は横断歩道の縞々が8本
  • ポストが黄色なのは昔はその色の馬車で郵便を運んでいたため(from 『スペインの黄色いポストの秘密』)
  • 道路の線に黄色やオレンジが使用されていない

    州・地域の見分け方

  • 環状道路と高速道路には都市を識別する文字が先頭についている。例）BIならばビルバオ付近
