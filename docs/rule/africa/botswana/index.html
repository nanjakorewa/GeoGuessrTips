国の見分け方

  • 車は左側通行
  • ドメインは.bw
  • 黄色と黒の色のポールが多い
  • 道路はざらざらした表面で、路側帯はオレンジ（無い場所もある）
  • ナンバープレートは白・黄・青が混在しているが、バンには青色が多い（気がする）
