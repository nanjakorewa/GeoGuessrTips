セネガル

国・地域の見分け方

  • ドメインは.sn
  • 言語はフランス語とウォロフ語が使われる
  • フランスと同じく通り名に「RUE」と書かれている
  • フランスと同じく右側通行
  • フランスで使用されるボラードや電柱が見つかる
  • ナンバープレートは全体が青色か左端だけ青色が多い
  • トラックの屋根が映っている場所が多くその種類で地域まで分かる(by Geotips )
  • 2015年に撮影されたものは空にひびのような線が入っていることが多い

見つかる標識

    ナンバープレートは全体が青色か左端だけ青色 。数は多くないものの、青色ナンバーは ボツワナもありえる 点に注意する。

    By BasilLeaf - Own work, CC0, Wikimedia Commons(Link)

    フランスとおなじ形の電柱 やボラード がある。また通り名に「RUE」と書かれていることも。1960年にフランスから独立するまではフランス植民地だった (参考文献 アフリカ分割)。

    By Princesssenegal - Own work, CC BY-SA 4.0, Link

    郊外に藁ぶき屋根の家が見つかることも 。

    セネガルや ボツワナ・ マダガスカルのようなサバンナ地帯ではバオバブの木が見つかる 。

    Google Carの屋根が見えたり Google Carを隠すためのモザイクがかなり大きいときがある 。ルーフレールが見えることもある 。

    2015年に撮影されたものに亀裂が入っていることが多い。亀裂や車が映っていない場所も多いので注意。

    フランスにある電柱とおなじ形の電柱がある。左がセネガル・右がフランス。

    これら以外にも色々な種類があるが、見た目は フランスのボラードに似ている。

    藁ぶき屋根の家やトタンの家が郊外でたまに見つかる 。

    道端に火を燃やしたあとが多いので調べてみたところ、ピーナッツを焼いて食べることが多いらしい。ただし道端にある燃やした跡がピーナッツによるものかは不明 (参考文献 落花生の脱穀 in Senegal)

    州・地域の絞り込み

    • 気候が南部と北部で異なり南部は熱帯・北部は乾燥した気候
    • 市外局番が33-8XXとなっているならばダカール
    • トラックの屋根が映っている場所が多くその種類で地域まで分かる(by Geotips )
    • 一番北の方は新品の電柱が多くコードが張られてないときもある

      南側でだけパームオイルの生産が多いらしい。ただし木自体はダカールのような場所にもたまに生えているっぽい。
