ナイジェリア

国・地域の見分け方

  • 言語は英語
  • 車は右側通行
  • ドメインは.ng
  • 黄色の小型タクシーが町を走っている
  • パトカーが付いてくることが多くランプを見ると地域もわかる(by Geotips )
  • GoogleCarのバーに黄色い縞模様がある時がある(by Geotips )
  • セブンアップが人気(参考文献 独立行政法人日本貿易振興機構(JETRO) アジア経済研究所 ナイジェリア - アフリカ成長企業ファイル)

見つかる標識

    黄色の小型タクシーが町を走っている

    パトカーが付いてくることが多い

    州・地域の絞り込み

    • 農業や植生が場所によって異なる
      • パームの木は海沿いの領域に多い
      • データ提供元：U.S. Department of Agriculture(USDA)・マップのサイズが一致するように一部画像加工あり
    • パトカーの赤青の位置で地域を絞り込める(by Geotips )
      • 内陸側・アブジャ周辺地域は赤色のランプが左側にある
      • 海沿いの地域は赤色のランプが右側にある
    • 通り名の看板の色などで地域を絞り込める(by Geotips )
