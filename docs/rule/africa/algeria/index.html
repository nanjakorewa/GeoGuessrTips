アルジェリア

国・地域の見分け方

  • 2023年11月の時点では公式カバレッジは無い
  • ドメインは.dz

    旧宗主国の フランスのものと近い形の電柱が見つかる。

    フランスの道路番号と同じく、黄色く小さいプレートに番号などが書かれていることがある。標識や路側帯に破線がありえるのもフランスと共通している。

    北部にはアトラス山脈が、南部にはホガール山地があり岩場が多い。画像はホガール山脈と思われる。地中海沿岸側は雨量も多く緑も見られる。
