国判定のルール

  • コロンビアのGoogleCarには黒い太目のアンテナがついている時とついていない時がある
  • 路側帯は白で、中央線は黄色
  • 標識の棒は細く白い素材で、看板の背面には白色か灰色のクロスが存在する
  • ナンバープレートは黄色
  • タクシーとトラックの屋根または側面にサイドプレートがあり、ナンバーと都市名が書いてある
  • ドメインは.co
