メキシコ

国・地域の見分け方

  • ドメインは.mx
  • 言語はスペイン語
  • OXXOはメキシコのコンビニ
  • 電柱は8角形のものが多い
  • メキシコの大手企業は～MEXや～MXと名前が付いていることが多い
  • 白色と黒色で塗られた円形のボラードがある
  • VeTVと書いてある青色のアンテナが家に付いていることがある
  • とまれが「ALTO」はメキシコ・ グアテマラ・ パナマ・ コスタリカ

見つかる標識

    ～MEXと名の付く会社はメキシコの大手企業に多い 。石油大手Pemex・通信大手telmex・セメント大手Cemexなど。OXXOはメキシコの大手コンビニで街中に多く見かける。ただしOXXOは チリの都市部にも店がある点に注意。

    8角形の電柱が特徴的 。ただし コロンビアにもたまにあって100%メキシコとはならないので周りの雰囲気も合わせてメキシコと判断する（コロンビアの例 ）。電柱に『～MEX』と書かれていたら確実にメキシコ 。

    とまれが「ALTO」 はメキシコ・ グアテマラ・ コスタリカ・ パナマのどれか。またメキシコの矢印はなんか太い気がする 。

    白色と黒色で塗られた円形のボラードがある 。

    メキシコのビール醸造所であるGrupo Modeloが製造するコロナビール (参考文献 コロナビール)のロゴが見られる 。

    州・地域の絞り込み

    • 市外局番の先頭の数字でおよその地域が特定できる(2~9)、偶数は上・奇数は下のイメージ
      • 55：メキシコシティ
      • 614：チワワ（Chihuahua）
      • 656：シウダー・フアレス（Ciudad Juárez）
      • 663・664：ティフアナ（Tijuana）
      • 686：メヒカリ（Mexicali）

      『55』はメキシコシティ。タクシーの模様も重要なヒント。

            都市・町の絞り込み

            • Holboxという町は本土から離れているため車が少なくゴルフカートが多い
            • ユカタン半島の東にある離島にCozumelという観光地がある
            • カンクン付近にIsla Mujeresという離島の観光地がありバギーが多い
            • Popocatépetl山付近の国立公園でGoogle Carが見えることがある

              ユカタン半島の北海岸にある島。希少な生物が多く動植物保護区に指定されている。環境保護の観点からホルボックスへの車両の進入は禁止されていて舗装もされてない。そのためゴルフカートやバギーのような車がほとんどで車は稀 (参考文献 Isla Holbox)。
