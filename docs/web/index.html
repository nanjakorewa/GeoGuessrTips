鉱山・資源 ジオゲッサーの始め方
geoguessr（ジオゲッサー）のプレイログを記録したページです。

GeoGuessr（ジオゲッサー）で下りた地点付近にあったものや攻略ヒントをブクマしてみるページ（ほぼ日記）です。世界に関するメモが多めです。記載されている会社名・商品・製品・サービス名・ロゴマーク等は各社の商標または各権利者の登録商標です。個人用のメモであり記載されている情報の正確さは保証していません。また書いてある説明やルールも完全に個人用のものなので、間違っていたり一般的では無かったりするかもしれません。左上の表示設定をクリックすると一部の文字を隠せます。記載されているtwitchやtwitterのユーザー名は配信とかで教えてくれた人です。ミスや問題点を見つけた場合はコメント欄・github issue・twitterのいずれかに連絡頂けるとうれしいです。
