Amerika Utara

Distribusi Pertanian

  • Jika ada jagung, lebih aman menuju ke wilayah tengah atau timur.
  • Sawah padi: jika tiang listrik memiliki garis kuning, kemungkinan besar di California utara; jika tidak, maka di sepanjang Sungai Mississippi.
  • Sumber data: Departemen Pertanian AS (U.S. Department of Agriculture)

    Jagung banyak diproduksi di sepanjang Corn Belt ( Corn: Production Acreage by County(USDA)). Namun, jagung diproduksi secara luas di seluruh negeri, sehingga ini saja tidak cukup untuk memastikan lokasinya di wilayah tengah.

    Distribusi kedelai mirip dengan jagung ( Crop Production Maps(USDA)).

    Sawah biasanya ditemukan di wilayah tertentu di California dan sepanjang sungai hingga pesisir Arkansas ( Crop Production Maps(USDA)). Petunjuk dari tiang listrik dapat membantu mengidentifikasi California.

    Pohon kapas jarang terlihat, tetapi ada kemungkinan menemukan serat kapas putih di jalan ( Crop Production Maps(USDA)).

    Tebu hanya tumbuh di wilayah tertentu di Florida, dan sangat sedikit di Louisiana karena terbatasnya akses jalan ( Crop Production Maps(USDA)).

    Wilayah gandum tidak tumpang tindih dengan wilayah jagung dan kedelai. Gambar gandum telah diproses agar memiliki bentuk peta yang konsisten dengan gambar lainnya.
