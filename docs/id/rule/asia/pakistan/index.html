Pakistan

  • Domain menggunakan .pk
  • Bahasa nasional adalah Urdu (اردو), sementara bahasa resmi adalah Inggris, tetapi banyak penutur bahasa lainnya
  • Pakaian tradisional shalwar kameez dapat ditemukan
  • Banyak Street View di fasilitas sekitar Lahore
  • Warna pelat nomor berbeda tergantung wilayah; di provinsi Punjab tempat Lahore berada, sisi kiri pelat nomor berwarna hijau
