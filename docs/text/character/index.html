言語

キリル文字

キリル文字はブルガリア語やセルビア語など多くの言語で使用されているが、字体は微妙に異なる。 キリル文字 - Wikipedia

台湾語

台湾（繁体字）と大陸側（簡体字）では同じ漢字の表記が違うことがある。

マレー語

東南アジアのマレー半島周辺地域で話されるオーストロネシア語族の主要な言語。地元の人たちの会話や看板などはほとんどがマレー語。 「di」＝場所や方向を示す前置詞、atのような扱い。「ke」＝同じく場所や方向だが遠くに向かう、toのようなイメージ？「dari」は逆に近づく感じ。

スペイン語

de（～の、～から）・la（冠詞、代名詞）・que（代名詞、接続詞）・los（冠詞、代名詞）・se（代名詞、再帰代名詞）が多く使用され、さらに「ñ」はスペイン語でのみ出現する。

フランス語

「â，î，û，ê，ô」のようなアクサン・スィルコンフレックス、「ç」「'」の表記はフランス語に多い。 「e」にアクサンテギュ（文字の上の記号）を付けることで発音が変わる。 使われる国はフランス、ベルギー、スイス、モナコ、ルクセンブルクなど。 アクサンテギュとは？フランス語の綴り字記号とアクセントを解説 フランス語圏 - Wikipedia

ポルトガル語

「ã」と「õ」はポルトガル語に特徴的な記号。たとえばサンパウロは「São Paulo」。ただし「ã」はベトナム語にもあるらしい。「Fone」はポルトガル語で電話の意味。 「ã」って何者？ ポルトガル語 - Wikipedia ベトナム語の文字についてるあの発音記号みたいなのは何？

ラトビア語

『Ļ』はラトビア語、ラトガリア語、リヴォニア語で用いられる。すべてラトビア周辺？ Ļ - Wikipedia

インドネシア語

発音が日本語のローマ字読みに近い読み方でいい。複数形を表現するためにハイフンで区切って言葉を繰り返す（島 pulau → pulau-pulau 島々）。 「TELP」は電話の意味。 インドネシア語 - Wikipedia

ヘブライ語

ヘブライ語は、アフロ・アジア語族のセム語派に属する北西セム諸語の一つ。ヘブル語、ヒブル語とも呼ばれる。 ヘブライ語 - Wikipedia

スロヴェニア語

スロヴェニアではリュブリャーナ大学などに日本語学科があったりして日本との交流もある。日本からは主に輸送機械、機械、電気機器類、鉄鋼製品を輸入している。 スロベニア共和国概況 令和３年６月現在在スロベニア日本国大使館 スロベニア語会話集
