    <h4 class="section-title">Como restringir o estado</h2>
    <p>Depois de decidir se está numa ilha, use as matrículas, os símbolos das estradas estaduais, o relevo e a vegetação, a planura do terreno e o estilo das casas para afunilar a região. Alguns estados oferecem pistas muito específicas.</p>
    <ul class="rule-list">
        <li>① Saber se os carros usam matrícula dianteira e as cores da placa ajuda a limitar o estado<a href="https://docs.google.com/spreadsheets/d/1P7LRYb13phYIYWlu4KSXxTosMGV44tHFU_EgqQvEXVQ/edit#gid=0">[Referência]</a>{{% by "rumilb" "twitch" %}}. Comece memorizando onde há chapa frontal.
            <ul>
                <li><a href="https://en.wikipedia.org/wiki/United_States_license_plate_designs_and_serial_formats">Lista de matrículas dos EUA (Wikipedia)</a></li>
                <li class="no-evidence">Estados com “new” no nome da placa tendem a ser amarelados{{% by "niwaisound" "twitch" %}}.</li>
                <li>Cores marcantes por estado
                    <ul>
                        <li>Placa praticamente preta → <span class="quiz">Delaware</span></li>
                        <li>Placa predominantemente verde → <span class="quiz">Vermont</span></li>
                        <li>Laranja escuro → <span class="quiz">Nova Iorque</span></li>
                        <li>Amarelo integral → <span class="quiz">Alasca</span></li>
                        <li>Amarelo pálido total ou nas extremidades → <span class="quiz">Mississippi</span></li>
                        <li>Fundo branco com faixa laranja à direita → <span class="quiz">Utah</span></li>
                        <li>Fundo branco com uma laranja desenhada → <span class="quiz">Florida / Geórgia</span></li>
                        <li>Fundo branco com árvore verde ao centro → <span class="quiz">Oregon</span></li>
                        <li>Fundo branco com faixa vermelha no topo → <span class="quiz">Ohio</span></li>
                        <li>Fundo branco com faixa azul-clara no topo → <span class="quiz">Minnesota / Montana / Iowa</span></li>
                        <li>Topo azul e base amarela → <span class="quiz">Pensilvânia</span></li>
                        <li>Topo castanho-avermelhado e base azul → <span class="quiz">Idaho</span></li>
                        <li>Topo levemente avermelhado e números em azul-escuro → <span class="quiz">Califórnia</span></li>
                        <li>Metade superior azul-viva e inferior branca → <span class="quiz">Connecticut</span>; <span class="quiz">Illinois</span> pode parecer similar.</li>
                        <li>Metade superior branca e inferior azul-clara → <span class="quiz">Kentucky</span></li>
                        <li class="no-evidence">Topo amarelado e base azul-clara → <span class="quiz">Washington?</span></li>
                        <li>Matrícula frontal com arco-íris → <span class="quiz">Havai</span></li>
                    </ul>
                </li>
            </ul>
        </li>
        <li>② Identificar placas de autoestradas interestaduais ou estaduais ajuda a localizar.
            <ul>
                <li>Os símbolos das estradas estaduais muitas vezes entregam o estado; em alguns casos o desenho replica o contorno do estado.</li>
                <li>As interestaduais (placa azul com “chapéu” vermelho) usam números pares para rotas leste–oeste e ímpares para norte–sul, com números maiores no nordeste.</li>
                <li>As saídas são numeradas (“Exit 1” etc.) e ficam visíveis ao ampliar.</li>
            </ul>
        </li>
        <li>③ As ilhas havaianas também têm Street View e exibem solos negros ou avermelhados{{% ex "https://goo.gl/maps/jtYyZC2QNiC829Hn6" "https://goo.gl/maps/P7nuAhYyR9LTpAxh9" %}}</li>
    </ul>
